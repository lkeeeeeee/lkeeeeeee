# XMD 실무 샘플소스
**본 레포지토리는 실제 프로젝트의 아키텍처와 흐름을 반영하여 제작된 샘플 소스입니다.**

**1. oclick-senddata-batch**

- C# (.NET Framework) WinForms 기반 배치 프로그램
- 외부 API 연계를 통해 매출 데이터를 전송합니다.

🔗 https://github.com/lkeeeeeee/oclick-senddata-batch.git  

**2. online-order**

- React 프론트엔드와 PHP 8.4(Laravel), Spring Boot 백엔드를 사용하는 멀티 백엔드 기반 웹 서비스 프로젝트입니다.
- 외부 쇼핑몰의 주문을 조회해 처리 한다. [ 주문수집, 주문조회, 송장출력, 송장검수, 판매등록, ERP 전산처리, 재고등록 ] 
- 카페24, 무신사, W컨셉, SSF샵, 11번가,고도몰, 롯데온, 카카오 등 다수 쇼핑몰 연동
- goodsFlow 송장업체 연동
- WMS (어시스트코리아 등) 연동

🔗 https://github.com/lkeeeeeee/online-order.git  

**3. kicc-tex-free**
* **보안 정책 및 기업 자산 보호**를 위해 현재 코드는 **비공개(Private)**로 전환되어 있습니다.
* 프로젝트의 핵심 설계나 구현 방식(핵심 로직)에 대해 궁금하신 경우, 면접 또는 개별 연락을 통해 안내해 드릴 수 있습니다.

- .NET(C#) 기반 DLL(Class Library) 활용 WinForms 애플리케이션
- kicc 텍스프리 개발대비 샘플

🔗 https://github.com/lkeeeeeee/kicc-tex-free.git

**4. mobile_receipt**

- Laravel(PHP) Framework & AngularJS 기반의 웹 애플리케이션
- 영수증 기반 매장 전자영수증 샘플
  
🔗 https://github.com/lkeeeeeee/mobile_reciept.git 

**5. qr-customer-bridge**
* **보안 정책 및 기업 자산 보호**를 위해 현재 코드는 **비공개(Private)**로 전환되어 있습니다.
* 프로젝트의 핵심 설계나 구현 방식(핵심 로직)에 대해 궁금하신 경우, 면접 또는 개별 연락을 통해 안내해 드릴 수 있습니다.

- Laravel(PHP) & AngularJS 기반의 QR 연동 고객 가입 및 인증 시스템
- QR Mapping: 고유 코드를 통한 매장/회원사 정보 동적 식별 및 맞춤형 가입 페이지 렌더링
- B2B Billing: 고객 가입 시 회원사 계정의 서비스 이용료(포인트) 실시간 차감 로직 구현

🔗 https://github.com/lkeeeeeee/qr-customer-bridge.git
